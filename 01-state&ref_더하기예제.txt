■ 변수처리
  ● return문안에서는 변수를 생성하거나 일반스크립트를 작성할 수 있음
  ● return문안에서는 변수나 함수를 사용시에는 {}를 사용

■ 함수와 이벤트 연동
  ● 객체의 속성이벤트를 사용

■state변수, useState
  ● 값이 변하면 화면이 변한다
  ● 함수를 임포트 해야 함
    import React, { useState } from 'react';
  ● 선언
    let [name,setName] = useState("홍길동")//let name = "홍길동"
  ● 변경
    setName(e.target.innerText)//name = e.target.innerText
