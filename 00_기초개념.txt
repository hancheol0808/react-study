■ SPA (Single Page App)
  ●여러페이지처럼 보이는데 사실은 한페이지로 구성됨
    - 페이지 이동시 시간이 걸리지 않는다
    - 설계가 어렵습니다.

■ PWA (Progressive Wep App)
  ● html,css,js로 앱을 설계할 수 있음

■ jsx문법
  ● html 유사구조를 이용하여 스크립트 문법을 작성할수있다.
  ● barbel 라이브러리가 설치되어 있어야 함.
  ● return 안쪽의 객체가 컴포넌트로 구성됨.
  ● return 안쪽의 객체는 한개의 부모객체로 감싸야 한다.
  ● return 안쪽에서는 if,for등의 일반 스크립트는 작성할 수 없음
  ● xml문법을 따른다. 단독객체는 닫기를 해야함<input />
  ● className, defaultValue, onClick,
